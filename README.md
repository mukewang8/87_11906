<h3>课程目标：</h3>
<p>规则引擎，全称为业务规则管理系统，英文名BRMS(Business Rule Management System)，它能够模仿人类的推理方式，并使用人类能理解的术语解释和证明它的推理结论。规则引擎实现了将业务决策从应用程序代码中分离出来，接收数据输入，解释业务规则，并根据业务规则做出业务决策。</p>
